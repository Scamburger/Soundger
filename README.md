# Soundger
Hello, folks! <img src="https://raw.githubusercontent.com/MartinHeinz//MartinHeinz/master/wave.gif" width="30px">
**Soundger** is a free analog of Spotify.
The project is currently under development

## Stack / Dependencies
* PostgreSQL
* Spring Web
* Spring Data
* Spring AOP
* Project Lombok
* Flyway
